LectureNoteCrawler
==================

Get all LectureNotes by Url!






How to compile it in console
--------------------------------------

```sh
$ javac ./src/com/goznauk/Main.java ./src/com/goznauk/HttpDownloader.java
```



* How to run in console
---------------------------------------
```sh
$ java com.goznauk.Main
```
